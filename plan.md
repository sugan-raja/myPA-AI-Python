# Plan: Integrating Ollama + Qwen 0.5B into Your Python Project

## 1. Prerequisites
- Install Ollama from https://ollama.com/download
- Pull the Qwen 0.5B model: `ollama pull qwen:0.5b`
- Ensure Ollama is running (default: http://localhost:11434)
- Install Python `requests` library: `pip install requests`

## 2. Add Ollama Integration Code
- Create a function to send prompts to Ollama and get responses.
- Example (in main.py or a new handler):

```python
import requests

def ollama_query(prompt, model="qwen:0.5b"):
    url = "http://localhost:11434/api/generate"
    payload = {"model": model, "prompt": prompt}
    response = requests.post(url, json=payload)
    if response.ok:
        return response.json().get("response", "")
    else:
        return "[Ollama error]"
```
