# Security Policy

## Supported Versions

We release patches for security vulnerabilities as needed. Please ensure you are using the latest version.

## Reporting a Vulnerability

If you find a security vulnerability in this project,please send email to sugan-raja@gmail.com
Do not open public GitHub issues for security-related matters.

Include the following details in your report:
- Description of the vulnerability
- Steps to reproduce (if applicable)
- Your contact information

We will respond as quickly as possible and coordinate a resolution.

Thank you for helping keep this project secure!
